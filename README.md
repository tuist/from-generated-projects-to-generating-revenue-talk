# 🎤 From Generating Projects to Generating Revenue

📝 Slides for a talk at **Deep Dish Swift 2026** about turning [Tuist](https://tuist.dev) from an open-source side project into a business.

## 🚀 Getting started

Make sure you have [mise](https://mise.jdx.dev/) installed, then:

```bash
mise install        # installs Node.js
npm install         # installs dependencies
npm run dev         # starts the Slidev dev server
```

## 📦 Building & exporting

```bash
npm run build       # builds a deployable SPA
npm run export      # exports slides to PDF
```
